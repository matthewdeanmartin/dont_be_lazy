# Installation

## Recommended (pipx)

```bash
pipx install dont_be_lazy
```

## pip

```bash
pip install dont_be_lazy
```

## From source

```bash
git clone https://github.com/matthewdeanmartin/dont_be_lazy.git
cd dont_be_lazy
uv sync --all-extras
```

## Verify the install

```bash
dont_be_lazy --help
```
