# Contributing

## Setup

```bash
git clone https://github.com/matthewdeanmartin/dont_be_lazy.git
cd dont_be_lazy
uv sync --all-extras
```

## Running checks

```bash
uv run make check
```

## Running tests only

```bash
uv run make test
```

## Before submitting a PR

```bash
uv run make prerelease
```
